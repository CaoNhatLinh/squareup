# Notification Sound Setup

This folder needs a `notification.mp3` file for order notifications.

## How to add notification sound:

1. **Option 1: Download a free notification sound**
   - Visit: https://notificationsounds.com/
   - Or: https://mixkit.co/free-sound-effects/notification/
   - Download an MP3 notification sound you like
   - Rename it to `notification.mp3`
   - Place it in this `public` folder

2. **Option 2: Use a built-in system sound**
   - Find a notification sound from your system
   - Convert it to MP3 if needed
   - Rename to `notification.mp3`
   - Place it here

3. **Option 3: Create a simple beep sound**
   - Use online tools like: https://www.audacityteam.org/
   - Generate a simple beep/tone
   - Export as `notification.mp3`
   - Place it here

## Recommended Sound Characteristics:
- Duration: 0.5 - 2 seconds
- Volume: Medium (not too loud)
- Format: MP3
- File size: < 100KB

After adding the file, the notification system will automatically play this sound when new orders arrive.
